٢. المعالجة والحسابات

لكل منتج عنده مخزون، بنحسب:

  • قيمة المخزون المجمّد = قيمة الـ quant، ولو مش متوفرة → الكمية × التكلفة.
  • أيام الركود = عدد الأيام من آخر بيعة لحد النهاردة.
  • الحالة: راكد فعلاً واقف من زمان ومفيش نمط موسمي → صفّيه · نائم موسمه جاي قريب → استنّى · انتهى موسمه موسمه عدّى ولسه عندك مخزون → صفّيه دلوقتي.
  • شرائح التقادم: ٦٠–٩٠ / ٩٠–١٨٠ / +١٨٠ يوم.
  • التجميع حسب الفئة والبراند + تحليل الموسمية لكل فئة.

٥. خطوات التحقق من الأرقام من Odoo

اختار منتج من جدول «المنتجات» في الداشبورد، سجّل قيمه، واتحقق منها واحدة واحدة:

١الكمية الموجودة (On Hand)
  1. Odoo → Inventory → Products → Products
  2. ابحث عن المنتج وافتحه، وشوف خانة On Hand
  3. ✓ لازم تساوي عمود «المخزون» في الداشبورد
٢قيمة المخزون (Value)
  1. Odoo → Inventory → Reporting → Valuation
  2. ابحث باسم المنتج وشوف عمود Value
  3. ✓ قارنها بعمود «القيمة» (520M = 520,000,000)
  4. لو صفر (مفيش Valuation) → الداشبورد بيحسبها = الكمية × التكلفة
٣تاريخ آخر بيعة وأيام الركود
  1. Odoo → Sales → Reporting → Sales (Sales Analysis)
  2. فلتر: Product = المنتج (طلبات مؤكَّدة)
  3. Group By → Order Date، رتّب تنازلي → أحدث تاريخ = آخر بيعة
  4. أيام الركود = النهاردة − تاريخ آخر بيعة
  5. ✓ لازم تطابق عمود «أيام ركود» (±يوم)
٤مبيعات آخر ٩٠ يوم
  1. نفس شاشة Sales Analysis
  2. فلتر: Product = المنتج + Order Date: Last 90 Days
  3. Measure → Quantity، شوف الإجمالي
  4. ✓ لازم يساوي عمود «مبيعات ٩٠ي»
٥الحالة (راكد / نائم / انتهى موسمه)
  • راكد فعلاً أيام الركود ≥ ٩٠ ومفيش نمط موسمي
  • نائم راكد دلوقتي بس الفئة بتبيع في الشهور الجاية (راجع خطوة ٨)
  • انتهى موسمه الفئة كانت بتبيع في الشهور اللي فاتت وخلصت

التحقق من الإجماليات

٦إجمالي قيمة المخزون
  1. Odoo → Inventory → Reporting → Valuation
  2. شيل الفلاتر → شوف إجمالي Value
  3. ✓ ده المقام المستخدم في «% من قيمة المخزون»
٧رأس المال المجمّد حسب الفئة / البراند
  1. Odoo → Inventory → Reporting → Valuation
  2. Group By → Product Category
  3. خُد فئة، واجمع قيم منتجاتها اللي مالهاش بيع حديث (خطوة ٣)
  4. ✓ لازم تقرب من «الراكد» للفئة في الداشبورد (تحقق بالعيّنة)
٨الموسمية (Heatmap)
  1. Odoo → Sales → Reporting → Sales
  2. فلتر: Product Category = الفئة + آخر سنة
  3. Group By → Order Date: Month و Measure → Quantity
  4. ✓ نفس شكل صف الفئة في الخريطة الحرارية (الأغمق = الأعلى)
طريقة سريعة للتأكد الشامل: اختار ٣ منتجات — راكد، نائم، وانتهى موسمه — وطبّق عليهم خطوات ١→٥. لو الثلاثة طابقوا، المنطق سليم وتقدر تثق في الباقي.
